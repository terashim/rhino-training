# ドキュメンテーションについて

パッケージと同様に、`box` モジュール内の関数に対しても `roxygen2` の文法でコメントを付けることでドキュメンテーションができる。`box::help` でこのコメントがパースされてドキュメントが表示される。

参考: [Get started • box](https://klmr.me/box/articles/box.html#writing-modules-1)
